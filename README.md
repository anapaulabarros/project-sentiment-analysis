# project-sentiment-analysis

O projeto propõe o desenvolvimento de uma aplicação de Inteligência Artificial para análise de sentimentos em avaliações de produtos. O problema consiste em classificar automaticamente comentários de consumidores como positivos ou negativos a partir de textos não estruturados, que podem conter ruídos, informalidade, erros de escrita, variações linguísticas e ambiguidades semânticas
